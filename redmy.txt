Квест по информатике

Задача игры:
Обезвредить бомбу, используя кодовое слово
После удачного обезвреживания бомбы игра запускается снова.
При каждом новом запуске игры генератор случайных чисел формирует новые исходные данные к задаче.

Игра написана на javascript. Для игры нужен только браузер

Задачу можно заменить на свою, поменяв картинку hepl.jpg  в папке img.
Далее в папке script открываем файл t_script.js,
в строчке
   let p=String(y*8); //код для бомбы 
Меням формулу 

Примеры ============
Задача 1:
n ребят обменялись рукопожатиями.Сколько было рукопожатий?
Число n ищите внизу страницы

Формула:
p=String(~~((y*y-y)/2));

Задача 2:
Дано число n. Переведите это число в двоичную систему.
Число n ищите внизу страницы.

Формула:
p=(y).toString(2);
